---
description: Promote a project-specific knowledge note to the global vault (available across all projects)
---

Use the `zed_promote` MCP tool to promote the note "$ARGUMENTS" from the project vault to the global vault.

Global knowledge is available across all projects — ideal for reusable patterns, anti-patterns, and learnings that apply everywhere.

If no note was specified, use `zed_hubs` and `zed_search` with tag "pattern" to suggest candidates for promotion.

After promotion, confirm the note is now in the global vault and searchable via `zed_global_search`.
